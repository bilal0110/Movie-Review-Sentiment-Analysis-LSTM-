# Movie Review Sentiment Analysis (LSTM)

## 💡 Overview
This project analyzes movie reviews and predicts whether the sentiment is **positive or negative** using a deep learning model.

---

## 🎯 Task
Binary classification of text:
- `Positive` 🙂  
- `Negative` 🙁  

---

## 📚 Dataset
- IMDB Movie Reviews Dataset  
- ~50,000 labeled reviews  
- Balanced dataset (positive & negative)

---

## 🧠 Model
- LSTM (Long Short-Term Memory) Network  
- Handles sequential text data effectively  

---

## ⚙️ Pipeline
1. Text preprocessing (cleaning, tokenization)  
2. Padding sequences  
3. Model training (LSTM)  
4. Evaluation & prediction  

---

## 🛠️ Tools & Libraries
- Python  
- TensorFlow / Keras  
- NumPy
